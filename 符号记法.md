# 数学符号记法说明（欧美标准）
本文档全面采用**ISO / AMS / LaTeX 欧美纯数规范**：变量与标量用斜体 `$x, y, f(x), \lambda$`，常数与标准函数名用正体 `$\mathrm{e}, \mathrm{i}, \pi, \ln, \log, \sin, \sinh$`（含自定义算子`\operatorname{}`），严禁斜体；微分、积分限与空集用`$\mathrm{d}x, \int_0^\infty, arnothing$`；数集与张量用`$\mathbb{R}, \mathbb{C}, \mathbb{N}, \mathbf{v}, \mathbf{A}$`；映射写作`$f: X o Y$`，定义用`$a := b$`或`$ riangleq$`；导数用`$f'(x), \dot{x}, \ddot{x}$`，偏导与梯度用`$\partial f/\partial x, abla f$`，积分写作`$\int_a^b f(x)\,\mathrm{d}x$`；线性代数中内积用`$\langle u, v angle$`，范数用`$\|x\|$`，转置用`$A^{\mathsf{T}}$`，共轭转置用`$A^{\mathsf{H}}$`；逻辑量词使用`对于所有，存在`，蕴含用`$\implies$`；特殊约定：**严禁使用`$\epsilon, \phi$`，强制改用`$ ε, φ$`**，且复数单位统一为`$\mathrm{i}$`。
