# 数学分析 (Mathematical Analysis)

> 以 **极限** 为核心，研究实数系上函数的 **连续性、微分、积分、级数** 的理论基础。

---

## 1. 极限 (Limits)
- 数列极限：$\displaystyle \lim_{n\to\infty}a_n=A$
- 函数极限：$\displaystyle \lim_{x\to x_0}f(x)=A$（$\varepsilon$-$\delta$ 语言）
- 关键：夹逼定理、单调有界定理、柯西收敛准则

## 2. 连续性 (Continuity)
- 定义：$\displaystyle \lim_{x\to x_0}f(x)=f(x_0)$
- 性质：闭区间上连续函数有界、取最值、介值定理

## 3. 微分学 (Differentiation)
- 导数：$\displaystyle f'(x_0)=\lim_{h\to0}\frac{f(x_0+h)-f(x_0)}{h}$
- 中值定理：Rolle、Lagrange、Cauchy
- 应用：单调性、极值、凸性、L’Hôpital 法则、Taylor 公式

## 4. 积分学 (Integration)
- 定积分：Riemann 和极限 $\displaystyle \int_a^b f(x)dx$
- 牛顿—莱布尼茨公式：$\displaystyle \int_a^b f(x)dx=F(b)-F(a)$
- 不定积分：原函数族
- 应用：面积、体积、弧长、微积分基本定理

## 5. 级数 (Series)
- 数项级数：$\displaystyle \sum_{n=1}^\infty a_n$（收敛判别：比较、比值、根值、Cauchy）
- 函数项级数 / 幂级数：收敛半径、一致收敛、逐项积分与求导

## 6. 常用定理速查
| 定理 | 要点 |
|---|---|
| 介值定理 | 连续函数取中间值 |
| 最值定理 | 闭区间连续函数有最大最小值 |
| 拉格朗日中值定理 | 平均变化率等于某点瞬时变化率 |
| 牛顿—莱布尼茨 | 沟通微分与积分 |
| 泰勒公式 | 局部多项式逼近 |

## 7. 与其它课的关系
- 高等数学：更偏计算与应用  
- 实变函数：推广积分（Lebesgue）、测度  
- 复变函数：复数域上的分析  
- 微分方程 / 泛函分析：依赖分析基础
